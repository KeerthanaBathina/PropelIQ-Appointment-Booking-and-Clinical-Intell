# User Story - US_057

## Story ID

- ID: US_057

## Story Title

- Staff Dashboard with Schedule and Tasks

## Description

- As a staff member, I want a dashboard displaying my daily appointment schedule, arrival queue summary, and pending tasks, so that I can manage my workday efficiently from a single view.

## Acceptance Criteria

1. **Given** the staff member logs in, **When** the staff dashboard loads, **Then** it displays today's appointment schedule, arrival queue count, and pending tasks (unverified codes, flagged conflicts, intake reviews).
2. **Given** the daily schedule is displayed, **When** appointments are listed, **Then** each shows patient name, time, type, status badge (color-coded), and no-show risk score.
3. **Given** pending tasks are displayed, **When** the staff member clicks a task, **Then** they are navigated to the relevant screen (SCR-014 for coding, SCR-013 for conflicts, SCR-012 for parsing).
4. **Given** the dashboard is loaded, **When** data changes occur (new arrival, status change), **Then** the dashboard updates within 5 seconds without manual refresh.

## Edge Cases

- What happens when a staff member has no appointments scheduled for the day? System displays an empty schedule with "No appointments today" and a link to the appointment management view.
- How does the system handle multiple staff members viewing the same patient changes? Real-time updates via cached views ensure all staff see consistent data within 5 seconds.
