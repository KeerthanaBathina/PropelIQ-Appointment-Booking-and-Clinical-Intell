# User Story - US_017

## Story ID

- ID: US_017

## Story Title

- Appointment Slot Viewing & Filtering

## Description

- As a patient, I want to view available appointment slots filtered by date, time, and provider, so that I can find a convenient appointment time.

## Acceptance Criteria

1. **Given** I am on the booking page, **When** I select a date range and optionally a provider, **Then** the system displays all available slots matching my criteria within 2 seconds.

2. **Given** available slots are displayed, **When** I view the calendar, **Then** dates with available slots show visual indicators (dots) and unavailable dates are grayed out.

3. **Given** I select a specific date, **When** the time slot grid loads, **Then** slots are shown in 30-minute increments with provider name and appointment type.

4. **Given** slots are cached, **When** I request slots for a date within the cache TTL (5 minutes), **Then** the response returns in sub-second time from Redis cache.

## Edge Cases

- What happens when no slots are available for my criteria? System displays "No available slots" with suggestions to try different dates or providers.
- How does the system handle booking windows beyond 90 days? Date picker restricts selection to 90 days from today per FR-013.
