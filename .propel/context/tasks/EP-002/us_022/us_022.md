# User Story - US_022

## Story ID

- ID: US_022

## Story Title

- Walk-in Appointment Booking by Staff

## Description

- As a staff member, I want to register walk-in patients for same-day appointments, so that unscheduled patients can be added to the queue and seen by a provider.

## Acceptance Criteria

1. **Given** I am on the staff dashboard, **When** I click "Walk-in Registration", **Then** a form opens for patient search or new patient creation with same-day slot selection.

2. **Given** a walk-in patient has an existing record, **When** I search by name, DOB, or phone, **Then** matching patient records are displayed for selection.

3. **Given** I select a same-day slot for the walk-in, **When** I confirm the booking, **Then** the appointment is created with "walk-in" designation and the patient is added to the arrival queue automatically.

4. **Given** no same-day slots are available, **When** the walk-in form loads, **Then** the system displays "No same-day slots available" with the next available date/time.

## Edge Cases

- What happens when a patient tries to book a walk-in? System blocks with "Walk-in bookings are available through staff only."
- How does the system handle an urgent walk-in when all slots are full? Staff can mark as "urgent" priority, which escalates to supervisor for emergency slot creation.
