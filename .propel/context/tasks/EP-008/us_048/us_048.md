# User Story - US_048

## Story ID

- ID: US_048

## Story Title

- AI CPT Procedure Code Mapping

## Description

- As a staff member, I want the AI to automatically map clinical procedures to CPT codes with justification text, so that procedure coding is accurate and ready for claims submission.

## Acceptance Criteria

1. **Given** clinical procedures have been extracted from patient documents, **When** AI coding runs, **Then** the system maps each procedure to the most appropriate CPT code with justification text.
2. **Given** a CPT mapping is generated, **When** the results are displayed, **Then** each code shows the CPT code, description, confidence score, and justification text.
3. **Given** multiple CPT codes apply to a single procedure, **When** the AI identifies this, **Then** the system presents all applicable codes ranked by relevance with multi-code assignment support.
4. **Given** the CPT code library is maintained, **When** a quarterly update is applied, **Then** the system refreshes the code library and revalidates pending codes.

## Edge Cases

- What happens when a procedure description is ambiguous? System assigns the closest match with reduced confidence and flags for staff verification.
- How does the system handle bundled procedures (multiple procedures sharing a single CPT code)? System identifies bundling opportunities and presents the bundled code option alongside individual codes.
