# User Story - US_047

## Story ID

- ID: US_047

## Story Title

- AI ICD-10 Diagnosis Code Mapping

## Description

- As a staff member, I want the AI to automatically map clinical diagnoses to ICD-10 codes with human-readable justification text, so that diagnosis coding is accurate, transparent, and ready for billing.

## Acceptance Criteria

1. **Given** clinical diagnoses have been extracted from patient documents, **When** AI coding runs, **Then** the system maps each diagnosis to the most appropriate ICD-10 code with a justification explaining the mapping rationale.
2. **Given** an ICD-10 mapping is generated, **When** the results are displayed, **Then** each code shows the ICD-10 code, description, confidence score, and justification text.
3. **Given** the ICD-10 code library is maintained, **When** a quarterly update is applied, **Then** the system refreshes the code library and revalidates any pending codes against the updated library.
4. **Given** multiple ICD-10 codes apply to a single diagnosis, **When** the AI identifies this, **Then** the system presents all applicable codes ranked by relevance.

## Edge Cases

- What happens when the AI cannot find a matching ICD-10 code? System assigns "uncodable" status with a confidence of 0.00 and flags for manual coding.
- How does the system handle deprecated ICD-10 codes after a library update? System flags existing records using deprecated codes and suggests replacement codes for staff review.
